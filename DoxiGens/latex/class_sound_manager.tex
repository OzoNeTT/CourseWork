\doxysubsection{Detailed Description}
Класс воспроизведения музыки. 

Данный класс служит для воспроизведения звуковых эффектов тех объектов, которые могут быть уничтожены. А также в данном классе применяется эффект \char`\"{}тряски\char`\"{} камеры.
